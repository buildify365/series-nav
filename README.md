# series-nav

티스토리 카테고리에서 자동으로 시리즈 목차를 가져오는 임베드 위젯.

## 사용법

[buildify365 시리즈 목차 생성기](https://www.buildify365.com/ko/widgets/series-nav)에서
URL 입력 → 디자인 선택 → 임베드 코드 복사 → 티스토리 글 HTML 모드에 붙여넣기

## 직접 임베드

```html
<div class="bf365-series" data-category-url="https://blog.tistory.com/category/시리즈명"></div>
<script src="https://cdn.jsdelivr.net/gh/buildify365/series-nav@main/series-nav.min.js" defer></script>
